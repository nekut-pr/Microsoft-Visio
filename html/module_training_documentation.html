module_training_documentation module¶

Модуль создания обучающих схем и инструкций с использованием Visio.

module_training_documentation.add_annotation_with_links(shape_id: str, resource_url: str) → None¶

Добавление аннотации с гиперссылкой на обучающие материалы. Реализация: встраивание интерактивных ссылок.

module_training_documentation.create_interactive_walkthrough(document: object, steps: list) → None¶

Создание интерактивного прохождения по диаграмме. Реализация: добавление навигационных кнопок, подсветка шагов.

module_training_documentation.create_tutorial_diagram(steps: list, screenshots: list = None) → object¶

Создание пошаговой обучающей диаграммы. Реализация: автоматическое создание последовательности с пояснениями.

module_training_documentation.embed_tooltips(shape_ids: list, tooltip_texts: dict) → None¶

Добавление всплывающих подсказок с обучающим контентом. Реализация: установка ScreenTip для фигур.

module_training_documentation.generate_user_manual(diagrams: list, output_format: str = 'pdf') → str¶

Генерация руководства пользователя из набора диаграмм. Реализация: сборка документов с оглавлением.
