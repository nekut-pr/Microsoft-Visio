module_gamification module¶

Модуль использования условного форматирования для отображения статусов задач и процессов.

module_gamification.add_progress_indicators(shape_ids: list, progress_field: str) → None¶

Добавление индикаторов выполнения (проценты, круговые диаграммы). Реализация: встраивание мини-виджетов в фигуры.

module_gamification.animate_status_changes() → None¶

Анимация при изменении статуса элемента. Реализация: добавление визуальных эффектов перехода.

module_gamification.apply_icon_sets(condition: str, icons: list) → None¶

Применение наборов иконок для визуальной геймификации. Реализация: выбор иконки по пороговым значениям.

module_gamification.set_status_formatting(shape_type: str, status_field: str, rules: dict) → None¶

Настройка условного форматирования в зависимости от статуса. Реализация: правила вида “если статус = Завершено, то цвет = зеленый”.
