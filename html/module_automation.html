module_automation module¶

Модуль использования встроенных инструментов Visio для автоматического обновления схем на основе данных.

module_automation.apply_rules_to_diagram(ruleset: dict) → None¶

Применение правил автоматической трансформации диаграммы. Реализация: выполнение скриптов изменения структуры при изменении данных.

module_automation.auto_update_on_data_change(data_source: str, refresh_interval: int = 60) → None¶

Настройка автоматического обновления диаграммы при изменении данных. Реализация: подписка на события изменения источника, триггер обновления.

module_automation.schedule_refresh(cron_expression: str) → None¶

Планирование периодического обновления диаграммы. Реализация: использование планировщика задач.

module_automation.trigger_auto_layout() → None¶
