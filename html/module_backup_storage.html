module_backup_storage module¶

Модуль сохранения файлов в облачных сервисах для предотвращения потери данных.

module_backup_storage.backup_to_onedrive(document_path: str, destination_folder: str) → bool¶

Резервное копирование документа в OneDrive. Реализация: загрузка файла через Graph API.

module_backup_storage.backup_to_sharepoint(document_path: str, site: str, library: str) → bool¶

Резервное копирование в SharePoint. Реализация: проверка прав, загрузка с версионированием.

module_backup_storage.restore_from_backup(backup_id: str, destination: str) → bool¶

Восстановление документа из резервной копии. Реализация: скачивание и замена текущей версии.

module_backup_storage.schedule_autobackup(frequency: str, target: str) → None¶

Настройка автоматического резервного копирования. Реализация: планировщик с конфигурацией частоты и места.
